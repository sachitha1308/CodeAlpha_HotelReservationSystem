# Hotel Reservation System

This is a console-based Java application developed as part of the CodeAlpha Java Internship.

## Features

* View available hotel rooms
* Room categorization (Standard, Deluxe, Suite)
* Book a hotel room
* Cancel reservations
* View reservation details
* Uses ArrayList and OOP concepts

## Technologies Used

* Java
* VS Code

## How to Run

1. Compile: javac HotelReservationSystem.java
2. Run: java HotelReservationSystem
